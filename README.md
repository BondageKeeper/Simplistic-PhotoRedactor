## 🛠️ Used tools(crucial libraries):
* **Pillow** - main tool to edit an image.
* **io+base64** - used for storing edited image in RAM memory.
* **flet** - represents the main Graphical User Interface (GUI).
* **time** - simply for calculating time.

## ⚡ Features:

### ✨ Image Filters
* **Blurring:** Softens the image details using a Gaussian-like smoothing filter.
* **Clarity:** Sharpens the edges to enhance fine image details.
* **Silvery & Edge:** Applies aesthetic monochrome styles and high-contrast edge detection.
  
### 📐 Transformations & Editing
* **Resize:** User can resize image or additional image.
* **Mask:** User can user black mask to cover something.
* **Rotation:** User can rotate an image.
